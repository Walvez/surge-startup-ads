上传到 GitHub 的推荐方法（Mac）：

1. 解压 surge-startup-ads.zip。
2. 打开“终端”，输入 cd 和一个空格，把解压后的 surge-startup-ads 文件夹拖进终端，然后回车。
3. 依次运行：

git init
git branch -M main
git add .
git commit -m "feat: initialize automatic Surge module builder"
git remote add origin https://github.com/Walvez/surge-startup-ads.git
git push -u origin main

如果提示远程仓库已有内容，先运行：
git pull --rebase origin main
然后再运行：
git push -u origin main

网页上传方法：
1. 解压 ZIP。
2. Finder 按 Command + Shift + . 显示隐藏文件，确保能看到 .github。
3. GitHub 仓库中选择 Add file → Upload files。
4. 把解压目录里的 .github、config、scripts、dist、README.md 一起拖进去并提交。

上传完成后：
1. 打开仓库 Actions。
2. 选择 Update selected Surge startup-ad module。
3. 点击 Run workflow。
